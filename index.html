Picsar - Photo Sharing App

Picsar is a user-friendly photo-sharing app that allows authenticated users to upload, share, and interact with images. The platform fosters social engagement through liking, commenting, and following other users. It empowers users to connect through visual storytelling, while providing an intuitive experience for exploring and interacting with content.

Key Features:

  • CRUD Functionality: Users can create, read, update, and delete their posts, allowing full control over their photo content.
  • Social Interaction: Like, comment, and engage with posts, creating an interactive experience within the community.
  • Search Functionality: Users can easily search for specific photos, hashtags, or users, making it simple to find relevant content.
  • User Authentication: Secure login with Google OAuth and JWT ensures a smooth and protected user experience.

Tech Stack:

  • Frontend: React.js, Redux, Redux-Thunk for state management
  • Backend: Node.js, Express.js
  • Database: MongoDB for storing user data and posts
  • Image Management: Cloudinary for handling image uploads
  • Authentication: JWT (JSON Web Tokens), Google OAuth
  • Styling: Material UI for a responsive and modern design

Chataroo - Real-time Messaging App

Chataroo is a real-time messaging application that enables users to communicate instantly via private or group chats. The app features a seamless messaging experience, with real-time notifications to keep users connected. Whether it’s for personal or group conversations, Chataroo ensures that messages are delivered with minimal delay.

Key Features:

  • Real-time Messaging: Users can send and receive messages instantly, making conversations feel immediate and uninterrupted.
  • Group Chats: Create and manage group chats for multiple participants, perfect for discussions, planning events, or casual hangouts.
  • Private Chats: Send direct, one-on-one messages for more personal conversations.
  • Notifications: Stay up-to-date with in-app notifications, including new messages, mentions, and updates in group chats, ensuring that you never miss a thing.

Tech Stack:

  • Frontend: React.js
  • Backend: Node.js, Express.js, Socket.io for real-time messaging
  • Database: MongoDB for storing user profiles and chat history
  • Authentication: JWT (JSON Web Tokens)
  • Styling: Chakra UI for a responsive, modern design
